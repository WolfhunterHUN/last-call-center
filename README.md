GAMEPLAY

Last Call Center is a first-person customer service simulator where you:

- Handle AI-driven apocalyptic support tickets
- Manage rising stress levels during your shift
- Interact with the office environment
- Use stress-relief items to stay functional
- Try to survive the shift without breaking down completely

Player decisions directly influence stress, pacing, and outcome.

TECHNICAL IMPLEMENTATION

CORE SCRIPTS OVERVIEW

InteractableObject.cs
Base interaction system handling all clickable objects (phone, monitor, items).

VideoNeocortexStarter.cs
AI conversation starter responsible for random customer prompt selection.

NeocortexScoreTracker.cs
AI response evaluator modifying stress based on POSITIVE or NEGATIVE actions.

StressManager.cs
Central stress system (Singleton) controlling escalation and Game Over at 100%.

StressBarUI.cs
Visual stress indicator using a color-coded animated bar.

StressReliefItem.cs
Defines stress relief items such as energy drinks, cigarettes, and coffee.

RandomDistantSounds.cs
Atmospheric background system using randomized 3D spatial audio.

MobilephoneAudio.cs
Phone voice effect with frequency filtering and distortion for realistic call audio.
